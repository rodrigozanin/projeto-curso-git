Principais comandos práticos do GIT - Curso Digital Innovation One 1- mkdir (make diretório / criar diretório no repositório local) 2- cd (change diretory / acessar um determinado diretório) 3- cd/ (volta para o menu raíz do diretorio) 4- git init (iniciar um repositório vazio, criando um diretório.git para começar a monitorar as alterações) 5- git status (mostra o estado atual do repositório, se existem arquivos sendo rastreados, alterações, exclusões) 6- touch (cria um arquivo dentro de um diretório) 7- git add (irá incluir o arquivo para ser monitorado pelo git) 8- git add (irá incluir todos os arquivos para ser monitorado) 9- git commit -m 'comentários' (identificar através de comentários a alteração da nova versão) 10- git log (exibe o histórico listando todos os com os commits informados) 11- git remote add origin (cria o link entre o git e o github para fazer as sincronizações) 12- git push -u origin master (realiza o teste 13- ls (mostra a lista de arquivos existentes no diretório que você está) 14- git remote rm orign (remove um repositório remoto) 15- git push -u origin master (iguala as versões do repositório remoto e local, fazendo todos os commits pendentes) 16- git branch (cria uma nova branch, mas permanece na que você estava) 17- git checkout (muda para a branch informada) 18- git checkout -b (cria a nova branch e já troca de branch) 19- git branch (lista todas as branchs criadas e qual está ativa) 20- git branch -D (deleta a branch informada...obs, você não pode estar com ela ativa) 21- clear (limpa o prompt do git)
